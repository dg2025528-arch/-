# 🌊 Ocean Biodiversity Analysis

## 프로젝트 소개

해양 환경 변화가 해양 종 다양성에 미치는 영향을 데이터 분석과 AI 예측을 통해 탐구하는 Streamlit 웹 애플리케이션입니다.

---

## 주요 기능

### 1. 해양 종 다양성 원인 분석

- 수온(SST)과 종 다양성 관계 분석
- pH와 종 다양성 관계 분석
- 해양열파 영향 분석
- 산호 백화현상 영향 분석
- 상관관계 히트맵 제공

### 2. 미래 해양 생태계 시뮬레이션

사용자가 미래 환경을 설정하면

- 예상 종 다양성
- 생태계 건강도
- 위험도

를 예측합니다.

---

## 데이터셋 변수

| 변수 | 설명 |
|--------|--------|
| SST (°C) | 해수면 온도 |
| pH Level | 해양 산성도 |
| Species Observed | 관측된 종 수 |
| Bleaching Severity | 산호 백화현상 수준 |
| Marine Heatwave | 해양열파 발생 여부 |
| Location | 관측 지역 |
| Date | 관측 날짜 |
