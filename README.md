# 🌊 Ocean Biodiversity Analytics

해양 환경 변화가 해양 종 다양성에 미치는 영향을 데이터 분석을 통해 탐구하고, 미래 해양 생태계 변화를 예측하는 Streamlit 기반 웹 애플리케이션입니다.

## 📖 프로젝트 개요

최근 지구온난화와 해양 산성화로 인해 해양 생태계가 빠르게 변화하고 있습니다.

본 프로젝트는 해양 환경 데이터를 분석하여 종 다양성 감소의 원인을 파악하고, 미래 환경 조건에서 해양 생태계가 어떻게 변화할지 예측하는 것을 목표로 합니다.

---

## 🎯 연구 질문

- 해양 종 다양성 감소의 주요 원인은 무엇인가?
- 수온 상승은 종 다양성에 어떤 영향을 미치는가?
- 해양 pH 변화는 종 다양성에 어떤 영향을 미치는가?
- 해양열파와 산호 백화현상은 생태계에 어떤 영향을 미치는가?
- 미래의 해양 생태계는 어떻게 변화할 것인가?

---

## 📊 주요 기능

### 1️⃣ 해양 종 다양성 원인 분석

- 종 다양성 분포 시각화
- 수온(SST)과 종 다양성 관계 분석
- pH와 종 다양성 관계 분석
- 해양열파 영향 분석
- 산호 백화현상 영향 분석
- 상관관계 분석

### 2️⃣ 미래 해양 생태계 시뮬레이션

사용자가 미래 환경을 설정하면

- 예상 종 다양성
- 생태계 건강도
- 환경 위험도

를 예측합니다.

## 📁 데이터 구성

| 변수명 | 설명 |
|---------|---------|
| SST (°C) | 해수면 온도 |
| pH Level | 해양 산성도 |
| Species Observed | 관측된 종 수 |
| Bleaching Severity | 산호 백화현상 수준 |
| Marine Heatwave | 해양열파 발생 여부 |
| Location | 관측 지역 |
| Date | 관측 날짜 |
