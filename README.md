## 📌 프로젝트 소개

대기 중 이산화탄소(CO₂)가 증가하면서 바다로 흡수되는 양도 늘어나고 있습니다.  
이 과정에서 해수의 **pH가 점점 낮아지는 해양 산성화** 현상이 발생하고 있으며,  
이는 산호 백화, 생물종 감소 등 해양 생태계 전반에 영향을 줄 수 있습니다.

이 프로젝트는 **2015~2023년 7개 주요 해역**의 관측 데이터를 분석하여,  
pH를 비롯한 여러 환경 요인이 **생물종 다양성에 미치는 영향**을 시각적으로 탐구하고,  
나아가 **미래 해양 생태계를 시뮬레이션**합니다.

---

## 🔬 연구 질문

> **해양 환경 요인(pH, 수온, 백화, 해양열파) 중 무엇이 생물종 다양성에 가장 큰 영향을 줄까?**  
> **그리고 이 추세가 계속된다면, 미래의 바다는 어떻게 변할까?**

## 📊 데이터셋

| 항목 | 내용 |
|------|------|
| **관측 기간** | 2015년 ~ 2023년 |
| **관측 지역** | 7개 해역 (그레이트 배리어 리프, 카리브해, 홍해, 몰디브, 갈라파고스, 하와이 제도, 남중국해) |
| **총 관측 수** | 약 500건 |
| **주요 변수** | 수면 수온(SST), pH, 산호 백화 심각도, 관측 종 수, 해양열파 발생 여부 |

## 🖥️ 페이지 구성

이 앱은 **두 개의 페이지**로 구성되어 있습니다.

### 📊 페이지 1. 분석 대시보드 (`Home.py`)

| 탭 | 내용 |
|----|------|
| 📉 **pH & 생태계 상관관계** | 연구 질문 → 데이터 탐색 → 요인별 분석 → 종합 결론 (보고서 형식) |
| 📆 **연도별 추세** | pH·수온·종 수·백화의 연도별 변화 추이 |
| 🗺️ **지역별 비교** | 7개 해역의 환경 지표 비교 및 레이더 차트 |
| 🪸 **백화 현상 분석** | 산호 백화 심각도 분포 및 원인 분석 |
| 🌡️ **해양열파 영향** | 해양열파 발생 시 생태계 변화 비교 |

### 🔮 페이지 2. 미래 시뮬레이션 (`pages/simulation.py`)

| 파트 | 내용 |
|------|------|
| 📈 **PART 1. 미래 추세 예측** | 2015~2023 추세를 2030년까지 연장하여 pH·수온·종 수 예측 |
| 🎚️ **PART 2. 직접 조작 시뮬레이션** | 슬라이더로 pH·수온·백화·열파를 바꿔 실시간 종 수 예측 (회귀 모델 기반) |
| 📊 **PART 3. 종합 비교** | 현재 평균 vs 2030 추세 예측 vs 내 설정값 비교 |

> 🤖 시뮬레이션은 **선형 회귀(Linear Regression)** 모델을 사용하며,  
> 사용자가 설정한 환경 조건에 따라 예상 생물종 수와 생태계 위험도(안전/주의/위험)를 보여줍니다.

## ⚠️ 시뮬레이션의 한계

이 프로젝트의 미래 예측은 **탐구 목적의 추정**이며, 다음과 같은 한계가 있습니다.

1. **선형 회귀 기반**이라 실제 데이터 범위를 벗어난 예측은 신뢰도가 낮습니다.
2. 9년치 데이터의 추세를 단순 연장한 것이라, 실제 미래와 다를 수 있습니다.
3. **상관관계는 인과관계가 아닙니다.** 여러 요인이 복합 작용하므로 단일 원인으로 단정할 수 없습니다.
4. 교육·탐구 목적이며, 실제 정책 결정용 예측이 아닙니다.

> 🎯 좋은 탐구는 완벽한 예측이 아니라 **정직한 한계 인정**에서 시작합니다.
